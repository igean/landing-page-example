Chrome 85 arrives with tab management

Google today launched Chrome 85 for Windows, Mac, Linux, Android, and iOS. Chrome 85 brings tab management changes, 10% faster page loads, PDF improvements, and a slew of developer features.

Faster page loads

Google is promising under-the-hood performance improvements with Chrome 85. You can expect two types of speed gains: Profile Guided Optimization, which delivers up to 10% faster page loads, and Tab Throttling, which helps reduce the impact of idle background tabs.

PDF improvements

Chrome 85 will let you fill out PDF forms and save them with your inputs. If you open the file again, you can pick up where you left off. The feature is rolling out over the next few weeks.

Tab management

If you use Chrome in tablet mode, there is now a new touchscreen interface with larger tabs that are more practical to organize and hide. The new look is coming to Chromebooks first — Google didn’t provide an exact time frame for this feature rollout.
